Q4.  What is Python?

Q5.  How memory is managed in Python?
Ans - Memory management in Python is handled automatically by the Python interpreter through a combination
of techniques, including reference counting and a garbage collector.

Q6.  What is the purpose continue statement in python?
Ans - The continue keyword is used to end the current iteration in a for loop (or a while loop),
 and continues to the next iteration.
